# Contributing to A-Society

A-Society is not currently accepting external contributions.

The project is in early development and the core framework design is still being established. Opening it to contributions before that foundation is stable would create more friction than value — for contributors and for the project.

If you have found a bug or have a concrete suggestion, opening an issue is the right move. Issues are read and considered even when pull requests are not.

This will change. When the project is ready to accept contributions, this file will say so clearly.
