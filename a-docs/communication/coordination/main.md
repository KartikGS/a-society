# A-Society: Coordination Protocols

This folder contains the standing rules that govern all inter-agent communication in the A-Society project. These rules apply to every unit of work, regardless of task or phase. They change only when the process changes — and that change requires deliberate review by the Owner.

---

## Documents

| File | What it governs |
|---|---|
| `handoff-protocol.md` | Status vocabulary, handoff requirements, receiver confirmation, clarification rules, pre-replacement checks |
| `feedback-protocol.md` | What to do when a discrepancy, false assumption, or blocker is discovered in prior work |
| `conflict-resolution.md` | Authority matrix, escalation path, human escalation threshold |

Read `handoff-protocol.md` first — the status vocabulary defined there is used by all other documents and all conversation artifacts.
